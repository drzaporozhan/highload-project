## Маркетплейс (Ozon-подобный сервис)

### Тема проекта
**Маркетплейс** — это онлайн-сервис для продажи и покупки товаров. Продавцы размещают товары, а покупатели заказывают их, выбирая по качеству, скорости доставки и цене.

### Существующие аналоги
- Ozon
- Wildberries
- Яндекс Маркет
- AliExpress
- Amazon

### Целевая аудитория

#### Местоположение аудитории

### Функциональные требования (MVP)

#### Ключевой функционал 
1. **Поиск и лента товаров:** категории, фильтры, сортировки.
2. **Просмотр карточки товара:** описание, цена, фото, начличие, срок доставки
3. **Избранное:** добавление/удаление, список избранных товаров.
4. **Авторизация и профиль покупателя:** аккаунт, адреса доставки, контакты.
5. **Чат покупатель-продавец**: сообщения, статусы прочтения, вложение изображения (минимально).
6. **Создание заказа:** выбор способа доставки/ПВЗ, подтверждение состава, фиксация итоговой цены.
7. **Модерация:** жалобы на товар, скрытие/бан, очередь модерации.

### Ключевые продуктовые решения 

- **Выдача товаров в ленте:** ранжирование с учётом базовой системы рекомендаций и фильтров/сортировок.
- **Поиск — отдельная подсистема**: товары индексируются в поисковом движке.
- **Медиа вне основной бд:** медиа файлы находятся в отдельном объектном хранилище, а в бд только ссылки и метаданные.
- **Эксроу для безопасной оплаты:** деньги удерживаются до подтверждения получения, и только после этого идёт выплата продавцу (или возврат).
- **Защита**: базовый антифрод с автоматическими парвилами, ручная модерация.
