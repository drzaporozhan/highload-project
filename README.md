# Расчётно-пояснительная записка для сервиса онлайн-рекрутинга

## Тема проекта
**Платформа онлайн-рекрутинга** — сервис, который соединяет соискателей и работодателей:
соискатели ищут вакансии и откликаются, работодатели публикуют вакансии и ищут кандидатов в базе резюме.

---

## Описание и аналоги

### Описание
Сервис онлайн-рекрутинга — это высоконагруженная платформа с двумя основными типами пользователей:
- **Соискатели**: создают резюме, ищут вакансии, откликаются, получают приглашения.
- **Работодатели/рекрутеры**: публикуют вакансии, получают отклики, ищут кандидатов в базе резюме

### Существующие аналоги
- hh.ru 
- SuperJob.ru
- Avito Работа

## Функциональные требования (MVP)

### Ключевой функционал
1. **Регистрация/аутентификация** и разделение ролей: соискатель / работодатель.
2. **Резюме**: создание/редактирование, хранение, настройки видимости (публичное/скрытое).
3. **Вакансии**: публикация работодателем, редактирование, архивирование, обновление.
4. **Поиск вакансий**: полнотекстовый поиск + фильтры (город, зарплата, график, опыт и т д), сортировки.
5. **Отклик на вакансию**: создание отклика, история откликов, статусы.
6. **Рекомендации/матчинг**: рекомендации вакансий соискателю и релевантных кандидатов работодателю (минимальная версия).


### Ключевые продуктовые решения
- **Два типа аккаунтов** с разными правами.
- **Резюме хранятся на сервере**, пользователь может **скрывать резюме** (оно остаётся в базе, но становится недоступным для поиска работодателями).
- **Контроль доступа к персональным данным**: контакты и детали резюме показываются строго по правилам (например, после отклика/приглашения или по настройкам видимости).
- **Поиск и ранжирование**: полнотекстовый индекс + бизнес-правила ранжирования (свежесть, релевантность, соответствие фильтрам).
- **Модерация и антифрод**: лимиты на действия, проверка вакансий/резюме, автоматические правила + ручная модерация.
